# Projet Zoo 

Ce projet est un mini système de gestion de zoo développé en Python dans le cadre du cours de Programmation Orientée Objet.

## Objectifs du projet

- Pratiquer la POO en Python
- Utiliser l’héritage, le polymorphisme et la composition
- Créer un menu interactif sans utiliser `break`
- Implémenter une classe `Vétérinaire` capable d’interagir avec les animaux
- Organiser le code en modules
- Préparer le projet pour devenir un package Python installable

## Classes principales

- **Animal** — classe mère avec nom, poids, taille
- **Souris**, **Serpent**, **Oiseau** — classes filles avec comportements spécifiques
- **Vétérinaire** — examine, pèse, mesure et traite les animaux
- **Zoo** — contient une liste d’animaux, permet d’ajouter et filtrer
